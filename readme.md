# 天气APP

天气助手app是我用来练习使用Jatpack组件开发的Android项目，api使用聚合数据中的免费api，应用有一个MainActivity，主界面通过BottomNavigationView导航三个Fragment，分别是天气、笑话、我的，天气界面使用ViewPag2，主要显示最近天气情况、生活指数等信息，笑话界面和我的界面正在开发中。
